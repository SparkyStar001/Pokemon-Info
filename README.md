# Pokémon Info Script

A simple Python script that fetches data from the **PokéAPI** and displays:

* Pokémon name
* ID
* Height & weight
* 4 random moves
* 2 random abilities

## Requirements

* Python 3
* `requests` package

Install dependencies:

```bash
pip install requests
```

## What This Script Does

1. Takes a Pokémon name as input.
2. Fetches Pokémon details from `https://pokeapi.co/api/v2/`.
3. Validates the name.
4. Displays basic Pokémon info.
5. Randomly selects and prints moves + abilities.

That's it — clean, simple, and beginner‑friendly.
